# Starbucks Website Clone

A clean front end clone of the Starbucks landing page built using HTML and CSS.
This project focuses on practicing layout design, color combinations, responsive structure, and creating real world UI sections such as the navigation bar, hero area, featured products, and footer.

---

## Features

* Responsive layout
* Clean and modern design
* Navigation bar inspired by Starbucks
* Hero section with promotional visuals
* Product cards similar to Starbucks items
* Well-structured spacing and alignment

---

## Tech Stack

* HTML
* CSS

---

## Project Structure

```
project folder
│── index.html
│── style.css
│── assets   (images used in the UI)
```

---

## How to Run

1. Download or clone the repository
2. Open index.html in any browser
3. View and explore the UI
